# Chatty Critter

A mobile-friendly web game where you adopt a cute AI-powered virtual pet, give it a personality, and chat with it!

**Made by:** Hanalulu Studio  
**Status:** Milestone 1 — Design & Setup ✅

## How to Open the Style Guide

1. Open the project folder in VS Code
2. Right-click `style-guide.html` → **Open with Live Server**  
   *(or just double-click the file to open in your browser)*
3. You'll see all the colors, fonts, and pet characters!

## Project Structure

```
Game-Chatty-Critter/
├── style-guide.html      ← Colors, fonts, and pet art preview
├── GAME_SPEC.md          ← Full game design document
├── PROJECT_PLAN.md       ← 8 milestones with tasks to check off
├── assets/

│   ├── items/            ← Shop items (coming in M7)

│   └── sounds/           ← Sound effects (coming in M8)
├── css/                  ← Game styles (coming in M2)
└── js/                   ← Game logic (coming in M3+)
```
