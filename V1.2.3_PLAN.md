## ✅ All Milestones Shipped

### M1 — Size & Position Overhaul
- **Dress-up wrapper enlarged** from 170×187 → **240×264px** (41% bigger)
- **XL size toggle** button: tap "🔍 Bigger" → 300×330px for fine placement detail
- All item positions **mathematically recalibrated** for 240×264 wrapper:
  - Head top ≈ top: 16%, eyes ≈ top: 35–38%, body center ≈ top: 72%, feet ≈ top: 88%
  - Center X of 240px wrapper = 120px → for 48px emoji: left ≈ 38–40%
- Rainbow accessory fixed: left: 20% (was 10% — off-screen on narrow devices)

### M2 — New Item Categories (3 new, 17 new items)

| Category | Items | Description |
|----------|-------|-------------|
| 🧚 **Wings & Back** | 6 items | Fairy, Butterfly, Angel, Devil, Dragon, Hero Cape — rendered BEHIND pet at z-index:0 |
| 👟 **Footwear** | 6 items | Sneakers, Boots, Ballet, Roller Skates, Flip Flops, Cozy Socks |
| 🧣 **Wraps & Body** | 5 items | Scarf, Bow Tie, Island Lei, Necklace, Knit Wrap |

Wings use `z-index: 0` (behind pet image at z-index: 1) so they appear on the pet's back naturally.

### M3 — 14 Backgrounds (was 6)

| New Backgrounds Added | Season/Mood |
|----------------------|-------------|
| 🍂 Autumn | Warm amber gradient |
| 🌅 Sunset | Orange-pink gradient |
| 🌈 Rainbow | 6-stop full spectrum |
| ❄️ Snowy | Ice blue → white |
| 🌸 Cherry Blossom | Pink gradient |
| 🌋 Volcano | Deep red-black |
| 🏖️ Beach | Sky-sand-water layered |
| ☁️ Cloud Kingdom | Soft sky blue |

### M4 — New Overlay Architecture

- **5 overlay spans** per pet wrapper (was 2):
  - `dressup-wings` / `home-wings` — behind pet (z-index 0)
  - `dressup-hat` / `home-hat` — front (z-index 3)
  - `dressup-accessory` / `home-accessory` — front (z-index 3)
  - `dressup-feet` / `home-feet` — front (z-index 3)
  - `dressup-wrap` / `home-wrap` — front (z-index 3)
- `applyOutfitToHome()` updated to render all 5 slots on the home screen
- `outfit` state now has 6 keys: `hat, accessory, wings, feet, wrap, bgId`

### M5 — UX Polish

- **Category tabs**: scrollable horizontal row with emoji icon + label, fits 6 categories
- **Item grid**: 4 columns (was 3), item emoji enlarged 26px → **32px**
- **Hover animation**: items lift and scale on hover with spring physics
- **Equipped glow**: gradient background + pop-in animation when item selected
- **Dressup-clear-btn**: styled as separate button with red hover (clear-all)
- **Size toggle**: `🔍 Bigger` / `🔽 Normal` button in overlay header
- Tabs auto-reset on close; size resets to normal on overlay close
