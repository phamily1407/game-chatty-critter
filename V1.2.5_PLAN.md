## Audit Findings (Code Review of v1.2.4)

### Bugs Still Present

| # | File | Issue | Severity |
|---|------|-------|----------|
| B1 | main.js `finishCreation` | New pet doesn't initialize `state.achievements` or `state.lifetimeStats` — returning player gets defaults but a reset player or new player on fresh session has undefined stats | HIGH |
| B2 | main.js `tickChallenge` | "Catch 10/20 balls" challenge ticks once per **game completion**, not per ball caught — players must play 10+ complete games to finish it | HIGH |
| B3 | main.js `fashionista` achievement | Logic adds `equippedCount` to lifetime total each save — saving 3 items once unlocks it correctly, but saving 1 item 3 times also unlocks it (wrong intent) | MEDIUM |
| B4 | main.js `tapPet` | Only 6 generic reactions, no species-specific flavour — feels identical every pet | MEDIUM |
| B5 | main.js `CHALLENGE_POOL` | No challenge for `dress` (dress up), `tap` (tap pet), `bubble` (bubble pop) — new actions don't contribute to challenges | LOW |

### Friction Points (Works but Feels Bad)

| Point | Where | Impact |
|-------|-------|--------|
| No first-visit guidance | Home screen | New players don't know to tap pet or hug it |
| Triple-tap trick not implemented | Home screen | Planned feature, expected by players but absent |
| Bubble Pop stub | Games screen | Memory Match added but no 3rd reflex game |
| Pet never contacts player unpromptedly after long absence | Chat | Emotional connection gap for returning players |
| Time of day has zero effect | Home | Game looks identical at 3am vs 3pm |

### What Players Would Notice in 15 Minutes
1. ✅ Hug button + floating hearts — delightful
2. ✅ Catch Ball combo + golden balls — exciting
3. ✅ Memory Match — satisfying
4. ❌ No tricks — "can I make it do tricks?" is a common expectation
5. ❌ Game looks same at night as day — feels static
6. ❌ After 15 min, games feel repetitive (only 2 active games)

## v1.2.5 Milestones

### M1 — Bug Fixes
- Fix `finishCreation` to initialize `achievements` and `lifetimeStats`  
- Fix `tickChallenge('catch')` to be called per ball caught (pass score into callback incrementally)
- Fix fashionista: check `equippedCount >= 3` at save time, store as boolean flag not accumulator
- Expand tap reactions: 5 generic + 3 species-specific per species = 38 total reactions

### M2 — Time of Day System
Home screen background and pet behaviour shift by local hour:

| Period | Hours | Background | Pet feels |
|--------|-------|-----------|-----------|
| 🌅 Dawn | 5–7 | Amber → peach | Stretchy, just woke up |
| ☀️ Morning | 7–12 | Sky blue → cream | Energetic, cheerful |
| 🌤️ Afternoon | 12–17 | Warm gold → green | Playful, active |
| 🌇 Evening | 17–20 | Orange → purple | Cosy, winding down |
| 🌙 Night | 20–5 | Deep indigo → black | Sleepy, gentle |

Night mode also slows the pet's idle bounce animation and adds subtle stars overlay.

### M3 — Pet Tricks (Triple-Tap)
Three taps within 1.5 seconds triggers a trick:
- **Cat**: Spinning pirouette
- **Dragon**: Fire-breath shimmy  
- **Puppy**: Excited zoom-around
- **Bunny / Rabbit**: Binkying (happy jump + twist)
- **Fox**: Sly tail-flip
- **Penguin**: Belly-slide
- **Parrot**: Wing-flap dance
- **Panda / Cloud Puff / Kangaroo**: Universal star-burst bounce

Each trick: unique CSS animation + `playTrick()` sound + species toast + happiness +5  
New achievement: "Show Off!" (do 10 tricks total)

### M4 — Bubble Pop Mini-Game
Canvas game, third mini-game option:
- Bubbles float up from the bottom at random positions
- Tap/click to pop before they escape off the top
- **Normal bubble**: 1 point (pastel)  
- **Golden bubble**: 3 points (gold glow) — appears every 8 pops  
- **Rainbow bubble**: 5 points + slow-time power (appears every 20 pops)
- **Bomb bubble**: avoid! (red, appears from level 5)
- Miss 5 bubbles → game over  
- Score × 2 = coins earned  
- Happiness +20, energy -8 on complete  
- Difficulty: starts slow, speeds up every 5 pops

### M5 — Weekly Pet Letter
Once every 7 days a ✉️ envelope appears on the Home screen. Tapping it:
1. Shows an animated letter-opening (envelope unfolds, paper slides out)
2. Displays a letter written in the pet's personality voice
3. Letter references: bond days, recent memory topics, current level, achievement count
4. After reading: envelope disappears for another 7 days, player earns +30 coins

Letter templates per personality (3 each = 15 total) reference state data dynamically.

### M6 — Polish + Ship
- New `playTrick()` sound in sound.js  
- CSS for: time-of-day backgrounds, trick animations, bubble pop overlay, letter overlay  
- HTML: bubble pop overlay, letter overlay, ✉️ indicator on home screen  
- Update CHALLENGE_POOL with dress-up and trick challenges  
- Commit + push v1.2.5

## Designer Vision for v1.3

After v1.2.5, the pet will feel **alive**. The next layer is making the world feel alive:

1. **Room Decoration** — buyable furniture overlays the home background (couch, plant, window, rug)
2. **Seasonal Events** — Valentine hearts in Feb, eggs in April, spooky in Oct — special foods, outfits, dialogue
3. **Photo Booth** — screenshot styled pet with branded frame → PNG download
4. **Friend Comparison** — local comparison (no server needed): export/import a "pet card" via clipboard link
5. **Voice Reactions** — Web Speech API: pet "speaks" its reply as text-to-speech (toggleable)
