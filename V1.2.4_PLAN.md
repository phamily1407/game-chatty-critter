## Bugs Found

### 🔴 BROKEN (will fail in production)

| # | File | Lines | Bug | Fix |
|---|------|-------|-----|-----|
| B1 | main.js | 1018 | `yKey` streak comparison uses `toISOString()` (UTC) but `todayKey()` returns local date — streak resets wrongly at UTC midnight | Calculate `yKey` using local date math |
| B2 | main.js | 247 | `state.friendsSince = todayKey()` stores a date string; later `new Date(state.friendsSince).getTime()` parses it as NaN → bond days always 0 | Store `Date.now()` timestamp, not string |
| B3 | games.js | 582 | Memory Match hard mode: 12 pairs = 24 cards, but `cols: 4` creates 6 rows — overflows mobile screen | Hard → `cols: 6`, Easy → `cols: 4`, Normal → `cols: 4` |
| B4 | main.js | 55 | `XP_PER_LEVEL[9]` is `undefined` → fallback 9999 XP for Level 10 — players can never reach it | Add entry: `2800` for level 10 |
| B5 | ai.js | 55-110 | No `AbortController` timeout on any API fetch — chat can freeze for 30+ seconds with AI hanging | Wrap each fetch with 12-second AbortController |
| B6 | main.js | 676 | Pet colour in Catch Ball uses hardcoded array index — breaks if species order ever changes | Replace with `{ cat: '#F4A7B9', puppy: '#A8D4EC', ... }` map |

### 🟡 WRONG (works, but wrong result)

| # | File | Bug | Fix |
|---|------|-----|-----|
| W1 | games.js | Catch Ball levels every 5 catches — players hit max difficulty in 40 catches (~4 min). Too fast. | Level every 8 catches |
| W2 | main.js | Hunger decay still 0.9/min — full hunger to critical in ~94 min. Players feel punished. | 0.5/min → ~185 min grace |
| W3 | main.js | `openMemoryMatch` plays `playLevelUp` on completion — wrong sound (level-up fanfare ≠ match win) | Use `playAchievement` or new `playGameWin` |
| W4 | main.js | Shop `shop-buy` button shows grayed state but no "Need X more" text — players don't know how far | Show "Need X 🪙" on all unaffordable items |
| W5 | main.js | Confetti in `runConfetti` uses fixed palette — doesn't match pet's species colour theme | Seed confetti palette from pet species colour |

### 🔵 MISSING (expected but not present)

| # | Feature | Impact |
|---|---------|--------|
| M1 | "Still waiting…" message after 5s of AI typing | Reduces AI-hang confusion |
| M2 | Achievement system — no badge collection anywhere | Major engagement driver |
| M3 | Memory triggers don't handle plurals/verb forms ("cookies" vs "cookie") | Reduces memory capture ~40% |
| M4 | Challenge rows have no visual progress bar | Progress invisible |
| M5 | No first-time Home tutorial hint | New players don't know to tap pet |

## Milestone Plan

### M1 — Critical Bug Fixes *(execute first)*
Fix B1–B6 + W1–W5 from table above.

### M2 — Achievement System *(biggest engagement add)*
10 achievements, unlocked by lifetime stats.  
Badges shown in Profile screen with glow animation on first unlock.

| Badge | Trigger |
|-------|---------|
| 💬 Hello World! | First chat message |
| 🗣️ Chatterbox | 50 chats |
| 🎾 Good Catch! | Catch 10 balls |
| 🏆 Ball Master | Catch 100 balls |
| 🃏 Memory Master | Complete Memory Match |
| ✨ Fashionista | Save outfit with 3+ items |
| 🔥 Loyal Friend | 7-day login streak |
| 🛍️ Shopaholic | Buy 5 items |
| 🤗 Big Hugger | Hug 10 times |
| 🪙 Coin Collector | Earn 500 total coins |

### M3 — Game Feel Polish
- Challenges: visual progress bar per task
- Memory triggers: add plural + verb variants (+40 words)
- Confetti matches pet colour
- AI chat: "Still thinking…" after 5s
- Shop: "Need X more 🪙" on all disabled buy buttons

### M4 — Stat & Economy Rebalance
- Hunger/affection decay further slowed (0.5/min)
- Catch Ball levels every 8 catches (was 5)
- Level 10 XP gap filled (2800 XP)
- Daily deal: weight towards more valuable items at higher level

### M5 — V1.2.4 Ship
- All fixes QA verified
- README updated
- Commit + tag v1.2.4 + push
